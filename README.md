## 2. 기획 의도
여행에서 필요한 서류나 일정 경비 등을 엑셀파일에 관리했었는데 모바일에서 조작하기가 다소 불편했음.<br> 
또한 일정을 시각적으로(구글 맵) 확인하고 싶은데 마땅한 방법을 찾지못함.<br>
이에 필요한 기능이 들어있는 어플을 만들기로 계획함.
